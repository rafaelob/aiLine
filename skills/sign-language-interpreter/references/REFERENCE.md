# Libras Grammar Reference

## Sentence Structure

Libras uses a flexible word order but commonly follows:
- **Topic-Comment**: The topic comes first, followed by the comment
  - Example: ESCOLA EU GOSTAR (School, I like)
- **OSV**: Object-Subject-Verb is common
  - Example: LIVRO EU LER (Book, I read)
- **SVO**: Also acceptable, especially in simple statements
  - Example: EU LER LIVRO (I read book)

## Non-Manual Markers (NMMs)

NMMs are essential in Libras — they modify meaning:

| Marker | Usage |
|--------|-------|
| Raised eyebrows | Yes/No questions |
| Furrowed eyebrows | Wh- questions (what, who, where, when, why, how) |
| Head shake | Negation |
| Head nod | Affirmation |
| Puffed cheeks | Large size/quantity |
| Pursed lips | Small size/precise movement |
| Eye gaze | Pronoun reference, spatial grammar |
| Body shift | Role shifting (quoting different speakers) |

## Classifiers (CL:)

Classifiers represent categories of objects by shape/movement:

| Classifier | Represents |
|-----------|------------|
| CL:1 | Person standing, thin object |
| CL:B | Flat surface, paper, wall |
| CL:C | Cup shape, cylindrical object |
| CL:V | Person walking, two-legged movement |
| CL:5 | Group of people, many items |
| CL:S | Holding small object |
| CL:F | Small round/flat object (coin, button) |

## Time Markers

Time is expressed at the BEGINNING of the sentence:
- PASSADO (past): slight backward shoulder movement
- AGORA (now): present tense, neutral position
- FUTURO (future): forward body lean or forward hand movement
- AMANHA, ONTEM, SEMANA-PASSADA, etc.: specific time signs

## Fingerspelling

Used for:
- Proper nouns (names, places)
- Technical terms with no established sign
- Abbreviations
- Marked with # prefix: #M-A-R-I-A

## Negation

Three common strategies:
1. Head shake during/after the sign
2. Specific negative signs: NAO, NADA, NENHUM, NUNCA
3. Opposite sign (e.g., GOSTAR vs NAO-GOSTAR)

## Questions

- **Yes/No**: Raised eyebrows + slight forward lean
- **Wh-**: Furrowed eyebrows + specific question sign (O-QUE, QUEM, ONDE, QUANDO, POR-QUE, COMO)
- Question sign typically at END of sentence
