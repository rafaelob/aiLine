# Multi-Language Content Adaptation Reference

## Language-Specific Guidelines

### Portuguese (Brazil) -> English (US)

| Area | Guideline |
|------|-----------|
| Formality | Brazilian Portuguese uses more formal register in education; adjust to US classroom norms |
| Grade levels | Ensino Fundamental I (1-5) ≈ Elementary; Fundamental II (6-9) ≈ Middle; Ensino Médio ≈ High School |
| Curriculum | BNCC → Common Core State Standards (CCSS) for ELA/Math, NGSS for Science |
| Assessment | Adapt "nota 0-10" to letter grades or rubric-based; ENEM context → SAT/ACT context |
| Cultural refs | Carnival → Thanksgiving/4th of July (as cultural touchpoints, not equivalents) |
| Measurement | Metric → Imperial with metric in parentheses for STEM |

### Portuguese (Brazil) -> Spanish (Latin America)

| Area | Guideline |
|------|-----------|
| Formality | Similar register; use neutral Latin American Spanish (avoid Spain-specific forms) |
| Grade levels | Similar structure; adapt to country-specific terms (grado, año) |
| Curriculum | Map BNCC to relevant country standards where available |
| False cognates | aceitar/aceptar, puxar/empujar, exquisito/exquisito (different meanings) |
| Vocabulary | Include regional variants for key terms (computador/computadora/ordenador) |

### English (US) -> Portuguese (Brazil)

| Area | Guideline |
|------|-----------|
| Formality | May need to increase formality for educational context |
| Idioms | Translate meaning, not words: "piece of cake" → "mamão com açúcar" |
| Measurement | Imperial → Metric |
| Cultural refs | Adapt US-specific references to Brazilian equivalents |

## Accessibility Translation Matrix

| Feature | PT-BR → EN | PT-BR → ES |
|---------|-----------|-----------|
| Alt text | Translate + cultural adapt | Translate + cultural adapt |
| Captions | Translate + timing adjust | Translate + timing adjust |
| Sign language | Libras glosses → ASL glosses | Libras glosses → LSM/LSE glosses |
| TTS script | Regenerate for target language | Regenerate for target language |
| Reading level | Adjust for English readability norms | Adjust for Spanish readability norms |
| Dyslexia fonts | Same recommendations apply | Same recommendations apply |

## Quality Metrics

- **Semantic fidelity**: Does the adapted content convey the same meaning? (target: >0.90)
- **Cultural appropriateness**: Are references, examples relevant to target culture? (target: >0.85)
- **Pedagogical alignment**: Are learning objectives preserved at same Bloom's level? (target: >0.90)
- **Readability**: Is reading level appropriate for target grade in target language? (target: within 1 grade level)
